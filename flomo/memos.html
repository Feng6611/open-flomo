#Openflomo/use

如何使用 Open-flomo：

  1. 导出你的flomo内容（根据官方信息，每7天只可以导出一次；如果你有不想公开的部分，可以在导出的时候提前筛选标签）
  2. Clone这个仓库
  3. 将解压后的flomo文件夹，导入当前项目
  4. 配置项更新：
    • `notesFileUrl: "flomo/memos.html"`：更新你实际的flomo文件路径
    • `minDate: "2024-01-01"`：仅显示该时间之后的memos
  5. 创建并提交代码，开启 Github Page
